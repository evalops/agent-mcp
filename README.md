# agent-mcp

Unified MCP server for external agents. A single MCP config line gives any
MCP-capable agent identity, governance, approvals, metering, and operating
rules from the EvalOps control plane.

## Source of truth

This repository is a **read-only mirror** of
`cmd/agent-mcp` and `internal/agentmcp` from
[`evalops/platform`](https://github.com/evalops/platform).

Do not open pull requests here. File issues and send patches against
`evalops/platform` — the mirror workflow re-publishes this repo on every push
to platform's `main` branch.
